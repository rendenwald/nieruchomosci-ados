## API Endpoint

```
GET /api/v1/properties/map
  ?bbox=minLng,minLat,maxLng,maxLat
  &zoom={zoomLevel}
  &propertyType={type}
  &auctionType={type}

Response:
{
  "markers": [MapMarker],
  "clusters": [MapCluster],
  "meta": { "total": number }
}
```

## Implementation Notes

- **Library:** MapLibre GL JS with `supercluster` for clustering
- **Clustering:** Server-side for initial load, client-side for dynamic zoom
- **Polygon drawing:** Use MapLibre GL Draw plugin
- **Filter sync:** Map filters ↔ URL query params ↔ results list (bidirectional)
- **Responsive:** Full height on mobile, split view on desktop (map left, list right)
